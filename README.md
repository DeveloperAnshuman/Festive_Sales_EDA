## 📌 Project Overview
This project performs an Exploratory Data Analysis (EDA) on a festive sales dataset to uncover trends, identify target demographics, and provide actionable insights for business growth. By analyzing purchasing patterns, we can optimize marketing strategies for future festive seasons.

## 📊 Key Objectives
* Identify the primary target audience (Gender, Age group, Occupation).
* Discover top-performing product categories and states.
* Analyze the correlation between demographics and purchasing power.
* Provide data-driven recommendations for inventory management.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Libraries:** * `Pandas`: For data cleaning and manipulation.
    * `NumPy`: For numerical computations.
    * `Matplotlib`: For static data visualization.
    * `Seaborn`: For statistical data visualization.

## 📂 Dataset Description
The dataset includes transaction details from a festive sale period. Key columns include:
* `User_ID` & `Cust_name`: Customer identification.
* `Product_ID` & `Product_Category`: Product classification.
* `Gender`, `Age Group`, `Marital_Status`: Demographic markers.
* `State` & `Zone`: Geographical data.
* `Orders` & `Amount`: Transactional metrics.

## 📈 Key Insights
* **Demographics:** Women aged 26-35 are the highest contributors to total sales.
* **Geography:** States like Uttar Pradesh, Maharashtra, and Karnataka lead in order volume.
* **Occupation:** Professionals in IT, Healthcare, and Aviation sectors show the highest purchasing power.
* **Product:** Food, Clothing, and Electronics are the most sold categories.
